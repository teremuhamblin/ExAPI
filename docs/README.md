# Documentation – ExAPI

Bienvenue dans la documentation officielle du projet **ExAPI**.

Ce dossier contient l’ensemble des guides, références techniques, bonnes pratiques et documents nécessaires pour comprendre, utiliser et contribuer efficacement au projet.

## Contenu

- **overview.md** — Présentation générale du projet  
- **installation.md** — Installation, configuration, environnement, Docker  
- **usage.md** — Utilisation de l’API, endpoints, exemples  
- **architecture.md** — Structure interne, modules, plugins, conventions  
- **development.md** — Workflow de développement, tests, contributions  

## Objectifs

- Fournir une documentation claire, complète et maintenable  
- Faciliter l’onboarding des contributeurs  
- Décrire l’architecture et les choix techniques  
- Centraliser toutes les informations utiles au projet

## Format

Chaque document est disponible en deux formats :
- **Markdown (.md)**
- **reStructuredText (.rst)**

## Mise à jour

Toute modification majeure doit être accompagnée :
- d’une mise à jour du document concerné  
- d’une entrée dans le CHANGELOG  
- d’une validation via Pull Request  

Bonne lecture et bonne contribution.
