# Overview – ExAPI

## Introduction
ExAPI est une API modulaire, performante et extensible construite avec **Bun**, **Fastify**, **TypeScript**, **Docker**, et une architecture pensée pour la production.

Elle fournit :
- une structure claire et maintenable
- un système de plugins
- une configuration flexible via `.env`
- une intégration Docker complète
- une documentation professionnelle

## Objectifs du projet
- Offrir une base solide pour créer des API modernes
- Simplifier le développement et la maintenance
- Permettre l’extension via modules et plugins
- Fournir une architecture propre et évolutive

## Fonctionnalités principales
- Serveur Fastify optimisé
- Routing modulaire
- Contrôleurs séparés
- Gestion d’erreurs centralisée
- Plugins internes et externes
- Support JWT
- Dockerfile + docker-compose
- Configuration multi-environnements
- Tests unitaires

## Public visé
- Développeurs cherchant une base API propre
- Projets nécessitant une architecture scalable
- Étudiants souhaitant comprendre une API moderne
- Entreprises voulant un socle stable et maintenable

## Technologies
- **Bun** (runtime ultra rapide)
- **Fastify** (framework performant)
- **TypeScript** (typage strict)
- **Docker** (déploiement reproductible)
- **PostgreSQL** (base de données)
